# 42subjects

## Overview
This repository summarizes the curriculum and projects from 42Tokyo.  
42 is a project-based programming school where students develop practical skills through hands-on assignments.

42Tokyoの課題を通じて、低レイヤからアプリケーション開発まで幅広い技術を習得しています。

---

## Curriculum Overview

42のカリキュラムでは、プロジェクトを通して以下の分野を段階的に学習します。

### Low-level Programming
- C言語による標準ライブラリの再実装（libft, ft_printf）
- メモリ管理・ポインタ操作
- システムコールの理解

---

### Algorithms & Data Structures
- ソートアルゴリズム最適化（push_swap）
- 再帰・バックトラッキング（迷路生成など）
- 計算量を意識した設計

---

### System Programming / UNIX
- プロセス管理・シグナル（minitalkなど）
- ファイル操作・入出力処理（get_next_line）
- シェル・OSレベルの理解

---

### Concurrent Programming
- スレッド・mutex・条件変数
- Dining Philosophers問題
- デッドロック回避設計

---

### Computer Graphics
- minilibXを用いた描画（fdf / cub3d）
- 3Dレンダリングの基礎

---

### System & Infrastructure
- 仮想マシン構築（Born2beroot）
- Linux環境設定・セキュリティ
- ネットワーク・サービス管理

---

### AI / Advanced Topics
- LLMのファインチューニング
- プロンプトから関数選択を行うシステム

---

## Key Features

- プロジェクトベース学習（座学なし）
- Peer-to-peer評価システム
- 実践的な課題設計
- 低レイヤから高レイヤまで網羅
